## Installation

1. Clone the repository:
  ```bash
  git clone https://github.com/yourusername/vidya_website.git
  ```
2. Navigate to the project directory:
  ```bash
  cd vidya_website
  ```
3. Install dependencies:
  ```bash
  npm install
  ```
4. Start the development server:
  ```bash
  npm start
  ```
